# booky_notes

***
### ``Description``
A CRUD web app to store notes about books.  
  
Built with:  
Node.js and Express.js  
Joi for form validation  
EJS for templating  
Bootstrap for styles and layout  
MongoDB for storage  
Mongoose as ODM
